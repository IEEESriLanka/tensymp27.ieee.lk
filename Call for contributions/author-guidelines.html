Author Guidelines

Home > Call for Contributions > Author Guidelines

Please read the following instructions carefully before submitting your paper. Failure to comply with these guidelines may result in rejection.

  1. Paper Layout and Format

    Papers must be written in English and formatted according to the IEEE two-column format. The maximum length is 6 pages, including figures and references.

    Download Word Template | Download LaTeX Template

  2. File Format

    All submissions must be in PDF format. Ensure that all fonts are embedded and the file is not password protected. The file size should not exceed 10MB.

  3. Blind Review Policy

    For the initial submission, please remove all author names and affiliations from the paper. Avoid self-referencing in a way that reveals your identity.

  4. Originality

    Papers must present original work. Content that has been previously published or is under review elsewhere is not acceptable.

  5. Submission System

    Papers must be submitted electronically via the online submission system linked below.
